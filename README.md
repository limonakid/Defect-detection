# Веб-приложение для обнаружения дефектов листового проката

Это демонстрационный проект по теме обнаружения дефектов поверхности стали с помощью YOLO.

## Что умеет приложение
- загружает изображение через HTML-форму;
- запускает детекцию через обученную модель `best.pt`;
- отображает изображение с bounding boxes;
- показывает класс дефекта и confidence.

## Структура проекта
- `app.py` — FastAPI-приложение;
- `templates/index.html` — HTML-интерфейс;
- `static/styles.css` — стили;
- `uploads/` — временные входные файлы;
- `results/` — выходные изображения с детекциями.

## Как запустить
### 1. Установить зависимости
```bash
py -m pip install -r requirements.txt
```

### 2. Подготовить модель
Создайте папку `weights` рядом с `app.py` и поместите туда файл `best.pt`.

Можно и без папки `weights`, если перед запуском указать переменную окружения:
```bash
set MODEL_PATH=C:\full\path\to\best.pt
```

### 3. Запустить сервер
```bash

py -3.8 -m uvicorn app:app --reload
```

### 4. Открыть в браузере
Адрес:
```text
http://127.0.0.1:8000
```

## Как это расширить до производственного уровня
- принимать кадры не вручную, а с промышленной камеры;
- сохранять результат в БД;
- добавить авторизацию оператора;
- интегрировать API с MES/SCADA;
- вести журнал дефектов и использовать его для дообучения модели.
